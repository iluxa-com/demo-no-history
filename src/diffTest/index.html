Gallery For Glass

Manage, Upload, and Create Vignettes with your pictures.

View project onGitHub

Features

Camera

The camera can be opened from this card in the applications list:

You can take a picture while my camera preview is visible by winking, taping, or clicking the camera button. If you two finger swipe to the right (away from you) the application will automatically jump to the gallery. Finally if you one finger swipe left or right the preview will zoom in and use that zoom magnification when you take the picture.

Gallery

The gallery can be opened from this card in the applications list:

While you are in the Gallery if you two finger swipe left (toward you) it will automatically switch to the camera. When you click a picture that is listed in the gallery you will get this list of options:


Vignettes

Vignettes are a feature that are already built into the standard Google Glass system. However, they only allow you to make it the second after you take the, and only with the card that you were looking at when you took the picture. For more info on how to make vignettes officially look Here.


Gallery for Glass allows you to create the same exact experience, but with pictures that you already on your device. You can sideload images for this system or just use images you have already taken! You can do voice input and place the text inside of the vignette.


Below are examples of the same exact vignette. However, the one on the left is the one generated by the official way to make it (right after it is taken) and the one on the right is created from my Gallery for Glass application.

Sending to Phone

The third option will upload the image to a server and be stored there. You then need to run the companion application (currently only available for Android) to download it to your phone. This feature will most like get replaced when Google Play Services are available and I can just enable uploading it to Google Drive. Downloading the image from your phone will require you to be logged into the same Google account glass it connected to. It will then download the image, and store it to the camera folder on your phone. If you select to delete the image off the server, then when it has finished downloading it will be permanently removed.


I opted for this option because I didn't want to be limited by only 1000 calls from the Mirror API.


Deleting

The final option that is available on each picture is delete. This allows you to permanently delete the photo off of your device without having to plug it into the computer.

How to Install

This application uses the Glass Development Kit, or GDK, which means it requires you to sideload it onto your Glass. Below are a couple videos that will walk you through the process of setting up the environment, and then how to sideload an application using the *.APK file.

Installing Drivers

This tutorial will show you how to install Google Glass drivers on Windows. If you are running MAC OS or Linux, you do not have to worry about this. For those running Windows 8, there is an additional step which is to Disable Driver Signature Verification.


NOTE: Enable Debug Mode before connecting to ADB. Swipe to Settings, Click on Device Info, and Turn on Debug Mode. This will allow communication between your ADB and Glass.


Sideloading APKs

This tutorial will show you how to Sideload (Install) and Uninstall any application to your Google Glass that are not listed under the MyGlass web page. For demonstration purpose Andrew will walk you through installing GlassCopter,a popular remake of a web based helicopter survival game.


Check your Glass connectivity:
adb devices
Install application:
adb install [application.apk]
List of packages:
adb shell pm list packages
Uninstall:
adb uninstall [package name]
